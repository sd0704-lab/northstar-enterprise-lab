## 06 – Communication Plan

## Purpose
To define how project information is communicated to stakeholders throughout the lifecycle of the Northstar Precision Manufacturing Enterprise Cyber Defense Lab.

## Communication Matrix
| Stakeholder | Information Required | Format | Frequency | Owner |
|-------------|---------------------|--------|----------|-------|
| Executive Sponsor | High-level progress | Executive Summary | Bi-weekly | Project Manager |
| IT Administrator | Technical configuration updates | Technical Report | Weekly | SysAdmin |
| Security Analyst | Detection performance metrics | Dashboard/Report | Weekly | Security Lead |
| Academic Advisor | Capstone progress | Email Summary | Monthly | Student PM |

## Escalation Path
1. Technical Issue → System Administrator  
2. Security Incident → Security Analyst  
3. Scope or Resource Impact → Project Manager  

## Documentation Storage
All project documentation is stored in the official GitHub repository under version control.
