# Project Charter  
Northstar Precision Manufacturing (NPM) – Enterprise Cyber Defense Lab

## Project Purpose
To design and implement a simulated enterprise cybersecurity infrastructure supporting blue team, incident response, and purple team operations.

## Project Objectives
- Deploy enterprise-style Active Directory environment
- Implement centralized logging and monitoring
- Establish security baseline configuration
- Develop detection use cases mapped to MITRE ATT&CK
- Document all infrastructure and security changes

## Stakeholders
- Scott Dempsey – Project Lead
- University of Phoenix – Academic Oversight
- Future Incident Response Team (Simulated)

## Success Criteria
- Domain services operational
- Logging and monitoring functional
- Security baseline implemented
- Environment fully documented
